# estudos-programacao
Repositório com meus estudos, práticas e projetos desenvolvidos durante minha jornada em programação.

# Conteúdo
- **Fundamentos**: primeiros programas em Java, variáveis e tipos.
- **Operadores**: cálculos e manipulação de valores.
- **Controle de Fluxo**: if, switch e lógica de aprovação.
- **Loops**: for, while, do-while, continue e break.

## Objetivo
Documentar minha evolução no aprendizado de programação e servir como portfólio para futuros projetos.
